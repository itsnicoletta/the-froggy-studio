Use this folder to keep project asset source files close to `data/projects.ts`.

Example:
- data/project-assets/theringexperience-cover.jpg
- data/project-assets/theringexperience-gallery-01.jpg

If you want images to be directly reachable by the browser without imports, keep the final
web-facing copies in `public/project-assets/` and reference them in `data/projects.ts`.
